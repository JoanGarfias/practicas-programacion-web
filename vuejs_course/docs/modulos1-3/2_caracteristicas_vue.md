# Qué es VirtualDOM

React fue el pionero del VirtualDOM y los demás frameworks lo utilizan ya que el VirtualDOM sirve mucho para aplicaciones grandes.

# CDN de VueJS

Se puede utilizar directamente el CDN de VueJS pero tiene limitaciones

# Web Component

También se puede utilizar VueJS 

# Single-Page Application (SPA)

Es la forma más común de utilizar VueJS. Un router interno simula que tenemos
distintas páginas pero en realidad es una aplicación que se renderiza en una sola página.


# Server-side Rendering

Se pueden agregar librerias y extensiones para Server-side Rendering. Aunque también hay una librería llamada Nuxt que provee las caracteristicas para trabajar con Server-side Rendering.

# JAMStack / Static Site Generation

Podemos utilizar Nuxt para generar un proyecto que provea todo el html estatico de nuestra aplicación para subirlo a un servidor como Github para desplegar la aplicación.
